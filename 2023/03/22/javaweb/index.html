# Maven

  • 标准的代码结构
  • 自动化项目构建方式
  • 方便的依赖管理

Maven: 构建和管理 Java 项目的工具

# 依赖

# 依赖传递

<dependencies>
    <dependency>
        <groupId>ch.qos.logback</groupId>
        <artifactId>logback-classic</artifactId>
        <version>1.2.3</version>
    </dependency>
</dependencies>
  • 直接依赖：当前项目通过依赖建立的关系
  • 间接依赖：依赖的项目中间接依赖的资源

# 排除依赖

排除依赖中的间接依赖

<dependencies>
    <dependency>
        <groupId>ch.qos.logback</groupId>
        <artifactId>logback-classic</artifactId>
        <version>1.2.3</version>
        <exclusions> 
        	<exclusion>
        		<groupId>xxxx</groupId>
        		<artifactId>yyy</artifactId>
        	</exclusion>
        </exclusions>
    </dependency>
</dependencies>

# 依赖范围

作用范围

  • 主程序
  • 测试程序
  • 是否参与打包程序

通过 <scope></scope> 设置其作用范围

# 生命周期

  • clean: 清理工作
  • default: 核心工作，如：编译，测试，打包，安装，部署等
  • site: 生成报告，发布站点
