# Html (上)

# the article from yogmas also sam

# 1. 基本架构

<html> 
<head></head>
<body></body>
</html>

# 2.html 标签

# 标题标签
<h1></h1>   ……  <h6></h6>

一共六级标题，文字加粗，由大到小依次减小。
大多数标签都是成对存在的，只有少数例如 br 的单标签

# 段落和换行标签
<p></p>
</br>

p 标签会自动换行，不用添加 br

# 文本格式化标签
<strong>加粗</strong> 或者 <b>加粗</b>
<em>倾斜</em> 或者 <i>倾斜</i>
<del>删除线</del> 或者 <s>删除线</s>
<ins>下划线</ins> 或者 <u>下划线</u>
# div 和 span 标签
<span></span>

形象来说 div 是一个大盒子，span 通常是用来放在大盒子里面的小盒子（后期 div 在页面构造中非常常用)

# 图像标签
<img src="路径" alt="图片不显示时显示出来的" 
title="鼠标放在图片上面显示出来的文字" 
width="" height="" border="">

在后期使用的 width,height,border, 都会放入 css 中来运行使用不会单独列入 img 中
图像标签的路径
1. 同一级路径
eg:(img.jpg)
2. 下一级路径
eg:(images/img.jpg)
3. 绝对路径
eg:(C:\Users\apple\Desktop\ 前端基础第一天 - HTML\ 案例 \img.jpg)
eg：(http://www.itcast.cn/2018czgw/images/logo.png)

# 超链接标签
<a href="链接">LOL</a>

LOL 可以以图片文字等等代替，点击就可以跳转到 href 所在的链接

可以添加的元素：target=“X”
X 包括_self (覆盖当前页面)
和_blank 重新启动新的页面
链接分类
1. 内部链接：存放在你网站前端的链接也就是你写前端时电脑里面的链接
2. 外部链接：例如 http 这类的其他网站的链接
3. 空链接：当你还没有链接可以填时，留出空位用 #代替
4. 下载链接：通常是以.zip 结尾的压缩包

# 锚点定位

由两个相互呼应的标签组成
可以从页面的一个地方跳转到当前页面的另外一个地方

<a href="#ID"></a>
<h3 id="ID">

ID 可以写成任何你定义的名称
h3 并不是固定的，可以换成任意标签
当你点击 a 中携带的元素或标签时你将会跳转到拥有 id=“” 且和 a 相同 id 的标签这

# 注释标签和特殊字符
<!-- 注释内容 --> 
<用&lt  >用&gt；(由于<>符号在html中的特殊性)

特殊内容可以用 ctrl+l 快速注释
