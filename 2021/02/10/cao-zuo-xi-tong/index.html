# 操作系统的发展史 (略)

# Unix

伴随着 B 语言和 C 语言的由来，同时由于其创作者可以直接访问使用该系统的电脑，电气公司将其私有化后随之产生了 Minix

# Minix

Unix 私有化之后，大学无法使用 Unix 系统进行教学，就出现了 Minix

# Linux

Minix 上可以实现的功能非常少，为了达到相应的功能需要，就出现了 Linux

# 创建和删除操作

  1. touch

    • 没有该名的文件存在，则创建一个新的文件
    • 有改名字的文件存在，则更改文件的末次修改日期

  2. mkdir

    -p 连续创建多个目录

    mkdir -p a/b/c/d

  3. rm

    -r 连续删除多个同一文件夹下的文件

    -f 强制删除无提示信息

    ls 的通配符对 rm 一样适用

操作系统
操作系统（operating system，简称OS）是管理计算机硬件与软件资源的计算机程序。操作系统需要处理如管理与配置内存、决定系统资源供需的优先次序、控制输入设备与输出设备、操作网络与管理文件系统等基本事务。操作系统也提供一个让用户与系统交互的操作界面。
